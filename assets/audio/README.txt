POWERPLAY — background music
============================

Drop ONE copyright-free looping track here, named exactly:

    theme.mp3      (preferred)
    theme.ogg      (also accepted)
    music.mp3      (also accepted)

It will loop through the whole game and automatically duck (quieten)
during each ball, then swell back. If no file is present the game is
simply silent except for the sound effects — nothing is synthesised,
so there is no background "scratch".

Where to get CC0 / royalty-free loops (free, no attribution needed):
  - https://pixabay.com/music/   (filter: "lo-fi", "chill", "casino",
    "acoustic"; all Pixabay audio is free for commercial use)
  - https://freesound.org/       (filter by License = "Creative Commons 0")
  - https://incompetech.com/     (Kevin MacLeod; CC-BY, needs credit)

Aim for a relaxed 1–3 minute loop that seamlessly repeats. Rename the
downloaded file to theme.mp3 and put it in this folder.
