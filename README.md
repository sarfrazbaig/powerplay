# Powerplay 🏏

A roguelike cricket **card duel** you play in your browser. Bowl an over, chase one, build your squad between matches, and lift the cup. Inspired by childhood cricket trump cards.

## ▶️ Play it

**https://sarfrazbaig.github.io/powerplay/** &nbsp;*(live once GitHub Pages finishes building — give it a minute after the first deploy)*

No install — it runs in any modern browser.

## How to play

1. **Bowl first.** Each ball, the rival reveals a batter. Pick one of your bowlers to answer it — a higher matchup leans the odds toward dots and wickets.
2. **Then chase.** In the second innings you bat; pick batters to beat the rival's bowlers and chase the target.
3. **Read the odds curve** before you commit — it shows whether your choice leans toward wickets/dots or boundaries.
4. **Win to earn coins**, then in the Clubhouse **sign new players, train, or release** to build a stronger squad for tougher matches.
5. Lose the chase and the run is over. Get further each time.
